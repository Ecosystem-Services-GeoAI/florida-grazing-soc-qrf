# GEE Environment Variable Scripts

This folder contains a collection of JavaScript (`.js`) scripts and reference links used to extract environmental variables from **Google Earth Engine (GEE)** for ecological modeling and geospatial analysis.

---

## 🌍 Description

Each `.js` file in this repository corresponds to a specific environmental dataset available in GEE, used to export region-specific geospatial variables. These scripts can be used in the [GEE Code Editor](https://code.earthengine.google.com/) by copying the content into a new script or importing the `.js` directly.

### Example Variables Included:
- Long-term average temperature (tmean, tmax, tmin)
- Monthly precipitation
- Vapor pressure deficit (vpdmin, vpdmax)
- Digital Elevation Models (e.g., USGS 3DEP 1m DEM)

---

## 🛠️ Usage

1. Open [Google Earth Engine Code Editor](https://code.earthengine.google.com/)
2. Create a new script and paste the desired `.js` code
3. Modify the **region of interest (ROI)** and **export settings** as needed
4. Run the script to export data to your Google Drive

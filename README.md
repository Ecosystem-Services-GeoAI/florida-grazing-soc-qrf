# FL-Grazinglands-SOC
This is a repository for the Florida grazing land SOC project. Total script or jupyter notebook consist of three part.

## 1. The environmental tiff download
This part includes javascripts to download envrionmental factors in the Google Earth Engine, such as soil moisture, vegetation indexes.

## 2. Point extraction from tiff with GPS location
This part mainly is about value extraction from tiff based on GPS locations from a csv file

## 3. Modeling with random forest
This part included script generating all figures in the paper, feature selection, training random forest, and make prediciton of SOC in Florida state.

# Modeling

This directory contains the notebook used for **model training, evaluation, data spiking experiments, and uncertainty quantification** in the soil organic carbon (SOC) mapping framework.

---

## Files

### Modeling.ipynb

This notebook implements the core modeling workflow used to estimate soil organic carbon (SOC) stocks and their associated predictive uncertainty.

The notebook includes the following main components:

1. Model Training and Evaluation

2. Data Spiking Experiments

3. Uncertainty Quantification and Validation

---

## Outputs

The modeling notebook generates:

- model performance metrics  
- evaluation results for different spiking strategies  
- prediction intervals for SOC estimates  
- uncertainty statistics used for subsequent spatial mapping

These outputs are later used to produce **SOC prediction maps and uncertainty maps** across the study region.

---

## Notes

All modeling steps are implemented in **Jupyter Notebook format** to facilitate transparency and reproducibility. The notebook can be executed sequentially to reproduce the full modeling workflow.
